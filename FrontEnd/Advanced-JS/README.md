# Advanced-JS

This folder contains the advanced JavaScript concepts I learned during my web development journey. Each file includes a simple example that I wrote while learning the topic. These examples are mainly for practice, revision, and strengthening my understanding of JavaScript.

## Topics Covered

- Asynchronous JavaScript
- Closures
- Promises
- Prototypal Inheritance
- `this` Context
- Async / Await
- ES6 Modules
- CommonJS Modules

## File Description

- `asynchronous.js` – Basic asynchronous JavaScript example using `setTimeout()`.
- `closure.js` – Example demonstrating JavaScript closures.
- `promises.js` – Promise creation and chaining examples.
- `prototypalInheritance.js` – Understanding prototype-based inheritance.
- `thisCotext.js` – Examples of the `this` keyword and different binding contexts.
- `asynAwait.js` – Async / Await examples using Promises.
- `math.js` & `app.js` – ES6 Modules example using `export` and `import`.
- `math2.js` & `app2.js` – CommonJS Modules example using `module.exports` and `require()`.
