- Constructor Functions
- Prototype Methods

- `instanceof`

## Quick Notes

### Constructor Function
Used before ES6 classes. Objects are created using `new`.

### Prototype
Methods placed on the prototype are shared by all objects, saving memory.

### Class
Cleaner syntax over constructor functions.

### Inheritance
`extends` creates a child class. `super()` calls the parent constructor.

### Encapsulation
Hide implementation details using private fields (`#field`).

### Polymorphism
Child classes override parent methods while keeping the same interface.

### Abstraction
Expose only what's necessary and hide internal logic.

### Getters & Setters
Control how properties are accessed and updated.
