# JavaScript Foundation

A beginner-friendly JavaScript foundation file for a GitHub Web Development roadmap.

## Topics
- Variables
- Data Types
- Functions
- Arrow Functions
- Conditionals
- Loops
- Arrays
- Objects
- Destructuring
- Spread & Rest
- Template Literals
- Optional Chaining

## Run

```bash
node foundation.js
```
